

Compile H264 codec using below steps:

1. H264 Decoder
   gmake -f Makefile(Windows)
   gmake -f Makefile_Linux(Linux)

2. H264 Encoder
   gmake -f Makefile.win32.h264
   gmake -f Makefile.linux.h264

Compile MPEG4 using below steps

1. MPEG4 Decoder
   gmake -f Makefile(Windows)
   gmake -f Makefile_Linux(Linux)

2. MPEG4 Encoder
   gmake -f Makefile.win32.mpeg4
   gmake -f Makefile.linux.mpeg4



Take the Input files for Encoding, decoding during execution from AV417_VIDEO_FILES directory
